Colombia Total Production Analysis

Training Phase

Based on the correlation and autocorrelation values for time series, we choose use machine learning time series algorithms such as: Linear Regression, Random Forest Regression, Optimized Distributive Gradient (XGBoost) and Long short-term Memory (LSTM) family of Recurrent Neural Network (RNN) architecture.

We perform the following steps:

1. Train test split: we separated our data in two parts, the test set and the rest of the data was used to train the models.

2. Scale the data: using a min-max scaler, we scaled the data so that all of our variables fall within the range of -1 to 1

3. Reverse scaling: After running the models, we used this helper function to reverse the scaling of step 2

4. Create a predictions data frame: generate a data frame that includes the actual data captured in our test set and the predicted results from the models so that we can quantify the best algorithm.

5. Score the models: this step uses the root mean squared error (RMSE) and mean absolute error (MAE) of our predictions to compare performance of our 4 models.

Process description

Model comparison phase

To compare the performance of the model, we analyzed the mean square error (RMSE) and the mean absolute error (MAE). Both measures are commonly used to compare model performance, but have slightly different intuition and mathematical significance

MAE: The mean absolute error tells us on average how far our predictions are from the true value. In this case, all errors are given equal weight.

RMSE: calculated through the square root of the sum of all squared errors. When we square, the largest errors have a greater impact on the overall error, while the smallest errors do not have as much weight in the overall error.

Process description

Forecast phase

Once we identified the best model for each country and database, that model was used to predict the data 13 years ahead.

Process description

Winning Forecast

Colombia will experience a U-shaped recovery right at the end of the decade after a predicted decreasing trend between 2020 and 2025 when the country is going to reduce its production by nearly 2 million bags.
